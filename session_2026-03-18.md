## Topics Covered

### 1. Codebase Exploration — Fuzzy Match Logic
Explored how T-Ragx retrieves and ranks translation memory (TM) segments.

**Two-stage retrieval pipeline:**
- **Stage 1 — BM25 (Elasticsearch):** Sends a `query_string` search against the source language field. Returns up to `top_k=10` candidates ranked by lexical relevance (term frequency × inverse document frequency). BM25 is bag-of-words — it ignores word order but rewards shared rare terms.
- **Stage 2 — Levenshtein reranking:** Re-sorts the 10 BM25 candidates by character-level edit distance (ascending). The closest string match rises to position 1 in the prompt. `normed_distance = distance / len(source_sentence)` is computed as metadata (0.0 = exact, higher = more different) but was NOT used downstream.

**Key files:**
- `src/t_ragx/processors/ElasticInputProcessor.py` — `rerank_elastic_result()`, `search_memory()`
- `src/t_ragx/models/BaseModel.py` — `trans_mem_to_text()`, `build_prompt()`

**No traditional TM brackets (75%, 85%, 100%)** — T-Ragx uses continuous signals, not tiered thresholds.

**Up to 10 TM segments** are passed to the LLM per sentence (default `top_k=10`). The `rerank_top_k=5` default in `batch_search_elastic` is overridden to 10 by `search_memory`.

---

### 2. Key Finding — `normed_distance` Was Unused
The Levenshtein score was computed per result but silently discarded. The LLM received all TM examples as equal-weight references with no quality signal, so a 99% match and a 40% match looked identical in the prompt.

### 3. New Feature Planned — Expose Similarity Score to LLM

**Goal:** Show `normed_distance` in the prompt so the LLM can weight closer TM matches more heavily.

**Two changes to `src/t_ragx/models/BaseModel.py`:**

#### Change 1 — `trans_mem_to_text()` (lines 27–35)
Add per-example score label in decimal format (0.0 = identical, 1.0 = completely different):

```python
def trans_mem_to_text(trans_mem: list, source_lang_code='ja', target_lang_code='en'):
    if len(trans_mem) < 1:
        return ""
    out_text = "Example translations (similarity: 0.0 = identical to source, 1.0 = completely different):\n"
    count = 1
    for row in trans_mem:
        normed_distance = row.get('normed_distance', None)
        if normed_distance is not None:
            score_label = f" [similarity: {normed_distance:.2f}]"
        else:
            score_label = ""
        out_text += f""" {count}.{score_label}\n   {row[source_lang_code]}\n   {row[target_lang_code]}\n"""
        count += 1
    return out_text
```

#### Change 2 — `build_prompt()` system message (lines 208–216)
Add TM score guidance unconditionally to the system message:

```python
system_content = (

if glossary_text:
```
